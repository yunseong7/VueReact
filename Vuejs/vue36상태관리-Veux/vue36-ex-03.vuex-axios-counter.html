외부 함수 호출 기능을 추가하기 위해서는 서버 기능이 필요합니다.

1. 실행 전에 서버를 실행하세요. Node서버실행

2. 브라우저로 서버 실행 여부 확인
http://localhost:5050/counter ==> 현재값 가져오기
http://localhost:5050/counter?step=1 ==> 1 씩 증가
http://localhost:5050/counter?step=10 ==> 10 씩 증가

예제 "vue36-ex-01.vuex-counter" 예제에 axios를 이용하여 외부 함수 호출 기능을 추가하여 본다. 이 앱의 특징은 다음과 같습니다. + 버튼을 클릭하면 카운터가 올라가고 - 버튼을 클릭하면 카운터가 감소됩니다. child-componet 템플릿 태그는 아래와 같이 설정하시오. <div class="component-child"> Child counter : {{ num }} <br> <button name="child" v-on:click="addCounter">+</button> <button name="child" v-on:click="subCounter">-</button> </div>
Parent counter : {{ counter }}
