npm 사용법 
      Vue에서의 Webpack 실습을 위해서는 Vue-CLI 를 사용해야 한다.
      교재 126페이지 참조.

      # CLI 는 command line interface의 약자.


      step 1. Vue-CLI 설치
            $ vue -V
            $ npm install @vue/cli -g
            $ vue -V
            @vue/cli 4.5.15

      step 2. Vue 프로젝트 생성
            $ vue  create  webpackprj
            Vue CLI v4.5.13
            ? Please pick a preset: (Use arrow keys)
            > Default ([Vue 2] babel, eslint)
              Default (Vue 3) ([Vue 3] babel, eslint)
              Manually select features

              ... 중략

              $ cd webpackprj
              $ npm run serve

            $ ls    # 현재 폴더 파일 목록 출력
            App.vue  fileload.png  footer.html  Footer.vue  header.html  Header.vue
            vue03-01.웹팩이해하기.html  vue03-03.파일합치기.jquery.html  vue03-05.파일합치기.Vue.html
            webpackprj/

      step 3. webpackprj 폴더 이동
            $ cd webpackprj

            $ pwd   # 현재 폴더 위치 확인
            /c/TIS/VueReact/Vuejs/vue03Webpack/webpackprj

            $ ls    # 현재 폴더 파일 목록 출력. package.json 이 있는지 확인
            babel.config.js  node_modules/  package.json  package-lock.json  public/  README.md  src/

      step 4. Vue 프로젝트 실행
            $ npm run serve

      step 5. 실습 파일 복사
            반드시 윈도우 탐색기에서 Header.vue, Footer.vue, App.vue 파일을
            webpackprj/src/ 에 붙여넣기
            visual studio code 에서 붙여넣기 하면 안됨.

      step 6. 터미널에서 실행중인 프로세스 종료
            "ctrl + c"  누르면 프로세스 종료 됨

      step 7. 운영 서버에 배포할 파일을 만든다. <== 웹팩 사용
            $ ls       # dist 폴더가 있는지 확인
            babel.config.js  node_modules/  package.json  package-lock.json  public/  README.md  src/

            $ npm run build    # webpackprj/dist/ 에 합쳐진 파일이 생성됨
