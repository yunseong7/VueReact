    node node -v
    npx npx -v 필요한 패키지를 설치하지 않고 해당 패키지를 찾아 실행 해주는 도구.
    npm npm -v 필요한 패키지를 설치 해주는 도구.
    yarn yarn -v

  • npm과 yarn 사용법
    설명 npm 명령어 yarn 명령어
    프로젝트 초기화 npm init yarn init
    프로젝트 운영 패키지 추가 npm install --save 패키지명 yarn add 패키지명
    프로젝트 개발 패키지 추가 npm install --save-dev 패키지명 yarn add --save-dev 패키지명
    전역 패키지 추가 npm install --global 패키지명 yarn add global 패키지명
    패키지 업데이트 npm update --save yarn upgrade
    패키지 삭제 npm uninstall 패키지명 yarn remove 패키지명
    패키지 전체 재설치 npm install yarn install
    캐시 삭제 npm cache clean yarn cache clean
    package.json의 명령 스크립트 실행 npm run 스크립트명 yarn 스크립트명
    react 프로젝트 만들기 npx create-react-app 프로젝트명 yarn create react-app 프로젝트명
    vue 프로젝트 만들기 npx vue create 프로젝트명 X
